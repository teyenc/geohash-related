Geohash Query Pipeline

"Find all 7-Elevens within 5 miles of me" — from API call to final result

1
User / GeoServer Request
The application receives the user's GPS location and search parameters.
API Call: Find "7-Eleven" within 5 miles of lat = 37.7749 lon = -122.4194
2
geohash_encode(lat, lon, 10)
Binary-search the Earth, alternating Lon/Lat splits. Each 5 bits = 1 Base32 character.
Bit 1 Lon: Is -122.4 >= 0.0? No → 0 World → West half Bit 2 Lat: Is 37.7 >= 0.0? Yes → 1 World → North half Bit 3 Lon: Is -122.4 >= -90.0? No → 0 West → Far-West Bit 4 Lat: Is 37.7 >= 45.0? No → 0 North → Lower-North Bit 5 Lon: Is -122.4 >= -135.0? Yes → 1 Far-W → East side Binary: 01001 = decimal 9 → base32[9] = '9' ... repeat for 10 characters ... Result: '9q8yyk8yut'
3
geohash_precision_for_miles(5)
Walk the cell-height lookup table. Pick the first level that fits inside the radius.
Level 1: 12430 mi <= 5? No Level 2: 1243 mi <= 5? No Level 3: 155 mi <= 5? No Level 4: 19.4 mi <= 5? No Level 5: 2.43 mi <= 5? YES → Pick Level 5 Truncate: '9q8yyk8yut' → '9q8yy' Steps: ceil(5 / 2.43) = 3 steps outward
4
Generate 7x7 Grid (geohash_adjacent walks)
From center cell '9q8yy', walk 3 steps N/S/E/W using string manipulation.
Yellow = center cell   Cyan = touches circle   Gray = corner (over-fetch)
Output: 49 strings → IN-list '9q8wx','9q8wz','9q8yn','9q8yp', ... '9q8yy', ... '9q9bb','9q9bc'
5
DocDB Index Scan (ix_mapdata3)
YSQL sends 49 exact-match lookups to DocDB. The md_name = '7-Eleven' filter is pushed down into the index scan (because md_name is the 2nd column of ix_mapdata3).
WHERE LEFT(geo_hash10, 5) IN ('9q8wx','9q8wz', ... 49 cells ...) AND md_name = '7-Eleven' DocDB scans LSM index on each TServer in parallel. Discards McDonald's, Starbucks, etc. at storage layer. Returns: 15 rows (only 7-Elevens)
6
YSQL Circle Filter (ST_DWithin / Haversine)
For each of the 15 rows, YSQL calculates the exact great-circle distance using Haversine. Rows beyond 5 miles (8046.7 m) are discarded.
ST_DWithin(geom::geography, user_loc, 8046.7) Row 1: 7-Eleven on Market St → 1.2 mi KEEP Row 2: 7-Eleven on Mission St → 0.8 mi KEEP Row 3: 7-Eleven on Geary Blvd → 2.1 mi KEEP ... Row 13: 7-Eleven on Daly City Ave → 4.9 mi KEEP Row 14: 7-Eleven on Pacifica Rd → 5.8 mi DISCARD (corner) Row 15: 7-Eleven on San Bruno Ave → 6.2 mi DISCARD (corner)
7
Final Result → Application / GeoServer
13 rows returned to the user. Each row contains the store name, address, coordinates, and distance.
