How B+ Trees Execute S2 Range Scans

The 2D map is divided into S2 cells. Each cell has an integer ID assigned along the Hilbert curve (0 to 15).

Those 1D integer IDs are inserted into the database's B+ Tree index as standard indexing keys.

Click the predefined query boxes below to see how a spatial bounding box translates into a single BETWEEN min AND max traversal across the bottom leaf nodes of the B+ Tree.
