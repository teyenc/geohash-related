2. The Query
SELECT * FROM places
WHERE tags && '{water, ski}';
Step 0: Setup
We have 4 rows. We create an index: CREATE INDEX ON places USING ybgin (tags).
Unlike a normal Postgres GIN which builds a local B-Tree, YugabyteDB's ybgin writes directly to DocDB (the distributed Key-Value store).
