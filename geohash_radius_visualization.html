Geohash Radius Search (geohash_in_list_within_miles)

When you query "find POIs within 5 miles", the database calculates how many Geohash cells it needs to step Outwards (North/South/East/West) to completely cover that radius.
It creates a square grid of cells (2*steps + 1) x (2*steps + 1). The actual distance check (ST_DWithin) later filters out the corners of this square.
