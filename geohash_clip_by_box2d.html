ST_ClipByBox2D — Sutherland-Hodgman

Clips a polygon to an axis-aligned bounding box by processing one edge at a time.
Each pass walks every polygon edge and keeps/discards/interpolates vertices against the clip boundary.

Sutherland-Hodgman edge rule (applied per edge):
StartEndAction
InsideInsideKeep start vertex
InsideOutsideKeep start + add intersection
OutsideInsideAdd intersection only
OutsideOutsideEmit nothing
ST_ClipByBox2D runs this 4 times (left → right → bottom → top). The output of each pass becomes the input for the next.
